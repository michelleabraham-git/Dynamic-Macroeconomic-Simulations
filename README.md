# Dynamic-Macroeconomic-Simulations
This repository contains analytical derivations and numerical simulations of classic macroeconomic dynamic models implemented in R.

Projects included:
Solow growth model – steady state derivation and stability analysis

Keynesian multiplier – difference equation dynamics and simulation

Cobweb price dynamics – naive vs adaptive expectations

Cobb-Douglas production function – neutral vs embodied technological change

The assignments combine mathematical derivations with R simulations to illustrate how macroeconomic systems converge to equilibrium under different parameter assumptions.
